# AI Game 儿童游戏网站项目记忆

> 3岁以内儿童个人游戏网站，持续更新中。

---

## 项目目标

为 3 岁以内幼儿打造一个安全、有趣、教育性的个人游戏网站。

---

## 设计原则

- 超大触控区域，按钮至少 80-100px
- 高对比度、鲜艳颜色（红、黄、蓝、绿为主）
- 点击即时反馈（声音 + 动画）
- 每屏只有 1-2 个交互元素，极简界面
- 无文字依赖，全图标/图片导航

---

## 游戏类型规划

| 类型 | 示例 | 核心技能 | 状态 |
|------|------|------|------|
| 点击出声音 | 动物叫声、乐器 | 因果关系 | 待开发 |
| 颜色/形状匹配 | 拖拽配对 | 认知分类 | 待开发 |
| 简单绘画 | 手指涂色 | 精细动作 | 待开发 |
| 数数游戏 | 点击计数 1-5 | 数字启蒙 | 待开发 |
| 泡泡/气球点击 | 点击消除 | 手眼协调 | 待开发 |

---

## 技术栈

- 纯 HTML / CSS / JS（无需复杂框架）
- Web Audio API 处理音效
- Touch events 优先，兼容鼠标
- 响应式设计，以平板为主
- PWA 支持离线使用

---

## 安全要求

- 无广告、无外链
- 无需登录或数据收集
- 防止误触退出（全屏模式）
