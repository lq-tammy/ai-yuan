## 设计原则

- 超大触控区域，按钮至少 80-100px
- 高对比度、鲜艳颜色（红、黄、蓝、绿为主）
- 点击即时反馈（声音 + 动画）
- 每屏只有 1-2 个交互元素，极简界面
- 无文字依赖，全图标/图片导航
- 视觉风格参考：鸟类主题 + 阿凡达魔幻森林（参考图 `birds.png`）

## 泡泡点点点游戏设计

**文件**：games/bubbles/index.html

**玩法**：
- 泡泡从底部飘起，点击爆炸得分
- 3种大小泡泡，18% 概率出现特殊泡泡（✨标记）
- 点中特殊泡泡 → 飞出鸟类 + 庆祝弹窗

**鸟类与泡泡对应**：
| 泡泡 | 鸟类 | 数量 |
|------|------|------|
| 小泡泡 (r=36) | 虎皮鹦鹉 🦜 | 3只 |
| 中泡泡 (r=56) | 八哥 🐦 | 2只 |
| 大泡泡 (r=80) | 鹈鹕 🦢 | 1只 |

**背景风格**：阿凡达魔幻森林
- 深邃墨绿色天空渐变
- SVG 巨型树冠剪影（多层）
- 生物荧光飘浮种子（Woodsprite）
- 薄雾层动画
- 荧光蘑菇 + 发光植物
- 背景环境鸟飞翔
- Web Audio API 合成 pop 音效（无需音频文件）

---

## 技术栈

- 纯 HTML / CSS / JS（无框架）
- Web Audio API 处理音效
- Touch / Pointer events 优先，兼容鼠标
- requestAnimationFrame 游戏主循环
- CSS 自定义属性驱动卡片主题色
- 响应式，以平板为主
